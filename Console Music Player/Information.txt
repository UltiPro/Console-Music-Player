--------------------------------------------------------------------------------------------------------------------------------------------------

Up Arrow -> Scroll Folder Up
Down Arrow -> Scroll Folder Down
Left Arrow -> Scroll File Up
Right Arrow -> Scroll File Down
Enter -> Choose Folder
Spacebar -> Choose File (Start)
F1 -> Mute
F2 -> Volume Down
F3 -> Volume Up
F4 -> Unmute
F5 -> Previous track (if the track lasts more than 2 seconds it starts again, otherwise it jumps to 
                      the previous track or the last one in the list if there are no more tracks to pass)
F6 -> Skip Back (10 seconds)
F7 -> Pause/Play
F8 -> Skip Forward (10 seconds)
F9 -> Next Track (if last, it goes to first in the list)
I -> THIS File
Escape -> Turn Off Application
